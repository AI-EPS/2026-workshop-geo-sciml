# 2026 Workshop Geo SciML

This repository contains the Julia workshop examples and notebooks for the 2026 Geo-SciML workshop.

For the workshop overview, schedule, and context, see:

https://ai-eps.github.io/workshops/2026.html

The numbered files are intended to be followed roughly in order, from basic Julia examples to simple neural-network and CNN demos.

To run the material, start Julia in this folder with:

```powershell
julia --project=.
```
